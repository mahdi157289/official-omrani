## 🎯 Project Goal

Transform **Makroudh Omrani** from a traditional pastry shop relying on Facebook/phone orders into a modern, scalable e-commerce platform that:
- Serves local Tunisian customers (Kairouan, Tunis)
- Reaches the global Tunisian diaspora
- Provides seamless online ordering experience
- Gives owners full control via admin panel

---

## 👥 Key Users

1. **Amira** - Local customer (Tunis), age 25-65, Arabic speaker, wants quick ordering
2. **Sami** - Diaspora customer (France), age 30-70, tech-savvy, wants to send gifts
3. **Mohamed** - Store owner, basic tech skills, needs analytics and order management

## 🎯 Core Requirements (Must-Have)

### 1. Multi-Language Support (FR-01)
- ✅ Arabic (RTL) - Primary
- ✅ French
- ✅ English
- Language switcher: Fixed bubble on right side
- Full RTL layout support

### 2. Guest Checkout with Auto-Account (FR-02)
- ✅ Order without creating account
- ✅ System automatically creates user account from order form
- ✅ Sends account claim email with token
- ✅ User can later claim account and set password

### 3. User Profile & Order History (FR-03)
- ✅ View past orders
- ✅ Edit personal information
- ✅ Manage addresses

### 4. Admin CMS with Real-Time Updates (FR-04)
- ✅ Payload CMS for content management
- ✅ Changes reflect live immediately (no manual refresh)
- ✅ Admin sidebar on right side
- ✅ Full control over products, categories, content

### 5. Admin Analytics Dashboard (FR-05)
- ✅ Sales overview (revenue, orders by time period)
- ✅ Top-selling products
- ✅ Order status distribution
- ✅ Customer metrics (new vs. returning)
- ✅ Date range filters

### 6. Enhanced Loading UX (FR-06)
- ✅ Initial loader: Black screen with "Makroudh Omrani..." text
- ✅ Skeleton screens for async data
- ✅ Smooth transitions

### 7. Cloudinary Media Integration (FR-07)
- ✅ Automatic image optimization
- ✅ CDN delivery
- ✅ Upload preset configuration

## 🛠️ Technical Stack

- **Frontend:** Next.js 15 (App Router), React 19, TypeScript
- **Styling:** Tailwind CSS, shadcn/ui
- **i18n:** next-intl
- **Backend:** Next.js API Routes, Payload CMS
- **Database:** PostgreSQL (Render), Prisma ORM
- **Auth:** NextAuth.js
- **Media:** Cloudinary
- **Deployment:** Render.com
- **Animation:** Framer Motion

---
